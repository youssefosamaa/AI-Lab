# AI Training Lab

This repository is a **personal AI training archive**.  
It contains multiple small-to-medium projects created while learning, practicing, solving guided projects, experimenting, and refining ML/DL concepts over time.

The main goal of this repo is **practice, documentation, and long-term reference**, not showcasing polished production systems.

---

## Purpose of This Repository

- Practice machine learning and deep learning concepts through hands-on projects  
- Document experiments, mistakes, and insights for future reference  
- Track learning progress over time  
- Maintain a clean and organized knowledge base  
- Revisit and improve old projects with new techniques
